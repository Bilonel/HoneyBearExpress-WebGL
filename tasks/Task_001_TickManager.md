# Task 001 - TickManager

## Goal
Create the simulation heartbeat.

## Requirements
- Create TickManager.
- Configurable tick rate (default 10 TPS).
- Expose Tick event.
- Start/Stop support.
- No gameplay logic.

## Do NOT
- Create Grid
- Create Buildings
- Create Items
- Create Economy

## Definition of Done
- Compiles
- Tick fires correctly
- Tick rate configurable
- Ready for future gameplay systems
